# 教授版摘要

## 文件用途

提供教授快速掌握本資料包的教學目標、學習價值、風險控管與評分方向。

## 目標讀者

- 教授
- 課程助教
- 需要審核教學設計的人員

## 章節目錄

1. 資料包摘要
2. Human-led Architecture 教學定位
3. 課堂導入方式
4. 工具作為例示而非唯一推薦
5. 安全、審查與測試要求
6. 學習成果與評分建議
7. 風險與限制
