# XML Prompt 模板

## 文件用途

整理課堂與專題可使用的 XML prompt templates，協助學生以結構化方式限制 Agent 任務範圍與輸出格式。

## 目標讀者

- 同學
- 助教
- 專題小組

## 章節目錄

1. 為什麼使用 XML semantic constraints
2. XML prompt 的基本結構
3. Inspect-only 模板
4. Plan-only 模板
5. Patch 模板
6. Debug 模板
7. Test generation 模板
8. Documentation 模板
9. 使用限制與風險
