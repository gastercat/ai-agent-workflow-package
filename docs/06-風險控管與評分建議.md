- 教授
- 助教
- 同學

3. Git/GitHub 審查流程
4. Branch 保護與 main 管理
5. Git diff review
6. Test harness 與驗證紀錄
7. Memory governance
8. Context window overflow 風險
9. 評分 rubric
10. 學術誠信與可追溯紀錄
