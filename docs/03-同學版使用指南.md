# 同學版使用指南

## 文件用途

提供同學實際使用 AI Agent 完成作業、專題與程式修改時的操作流程與注意事項。

## 目標讀者

- 修課同學
- 專題小組成員
- Git/GitHub 初學者

## 章節目錄

1. 使用前心態：人主導，Agent 輔助
2. 課堂與專題的基本工作流
3. Git 與 GitHub 基礎
4. 使用 Agent 前的準備
5. Inspect、plan、patch、debug、test、document 流程
6. 如何閱讀 Agent 的修改
7. 如何整理可審查紀錄
8. 常見錯誤與 edge cases
